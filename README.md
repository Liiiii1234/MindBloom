# 🌻 MindBloom: 基于大模型的大学生心理健康支持 Web 平台

> **项目愿景：** 针对大学生群体的心理健康痛点，探索“AI + 心理干预”的创新服务模式，提供全天候的专属情绪支持。

## 📖 项目简介
MindBloom 是一个全栈 Web 应用程序，专为大学生心理健康设计。平台不仅提供了结构化的情绪记录与专业测评功能，更深度接入了 DeepSeek 大模型 API，构建了一个具备上下文记忆、能提供共情与疏导的 AI 心理助手。本项目目前已在校内成功推广试用。

## ✨ 核心功能
* **💬 专属 AI 心理疏导 (AI Chatbot)：** 接入 DeepSeek API 并进行 Prompt 优化，AI 能够以“专业心理咨询师”的身份与用户进行多轮对话，提供实时的情绪支持。
* **📝 情绪日记 (Mood Tracking)：** 支持用户记录每日心情，平台进行数据持久化保存，帮助用户进行长期的自我情绪觉察。
* **📊 专业心理测评 (Assessment)：** 内置标准化心理健康问卷，通过量化数据辅助评估用户的心理状态。

## 🛠️ 技术架构与技术栈
本项目采用前沿的 Serverless (无服务器) 与 BaaS 架构设计：

* **前端 (Frontend):** `React` + `Vite` + `shadcn/ui` —— 构建高性能的响应式单页应用。
* **后端即服务 (BaaS):** `Supabase` (PostgreSQL) —— 实现开箱即用的用户认证与业务数据（问卷、日记）的云端持久化存储。
* **核心逻辑 (Serverless Backend):** `Supabase Edge Functions` —— 编写无服务器函数处理复杂计算与模型调用，安全隔离第三方 API 密钥。
* **大模型驱动 (AI Engine):** `DeepSeek API` —— 提供核心的自然语言处理与情感陪伴能力。
